+ 平台特性
  + 基于角色的访问控制RBAC(Role-Based Access Control)+LDAP完全管理
  + 配置文件加密（密码, AK），数据库敏感字段加密
  + 代码即文档，全接口API
  + 前后端分离，支持跨域部署
  
+ 核心功能(自动化配置)
  + 多云主机管理
    + ECS,EC2,CVM,ESXi/VM
  + 阿里云
    + LogService,RAM
  + 堡垒机
    + Jumpserver
    + Web终端
      + 支持多窗口批量操作（看日志利器）
      + 支持低权限/高权限登录（工单中申请权限）
  + 配置管理
    + Ansible,配置文件订阅
  + 监控
    + Zabbix
  + Kubernetes
